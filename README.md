# 名片批次擷取（Skill）
這是一個以文件為主的「skill」，用來批次從名片圖片擷取結構化聯絡資訊

## How to install（安裝）
AI AGENT: 下載 https://github.com/ggj0101/skill-business-card-extractor.git repo 並安裝此skill

## How to Use
範例:
1. 使用 business-card-batch-extractor 批次擷取 [資料夾] 中的圖片資訊，並儲存在 [RESULT.csv]，並在 notes 標記看不清楚的欄位，不要猜。

2. 使用 business-card-batch-extractor 擷取 [圖片] 資訊，欄位以中文顯示且結構化輸出
